# 🎬 Movie Recommendation System

An AI-powered Movie Recommendation Web App built using Flask, Pandas, and IBM Watson Assistant.

---

## 🚀 Features

- Recommend movies based on:
  - Genre
  - Language
- Clean and simple web interface
- Dropdown-based user input (no typing errors)
- IBM Watson Assistant chatbot integration
- Fast and lightweight

---

## 🛠️ Technologies Used

- Python
- Flask
- Pandas
- HTML
- CSS
- IBM Watson Assistant
